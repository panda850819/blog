## [2025-12-26] - Chiri 風格主題更新

### 改進
- 🎨 **主題升級為 Chiri 極簡風格**
  - 採用柔和黑色文字 `rgba(0, 0, 0, 0.85)` 取代純黑
  - 次要文字從 `rgba(0, 0, 0, 0.4)` 提升到 `rgba(0, 0, 0, 0.6)` 增強可讀性
  - 連結顏色改為與主文字一致的柔和黑色
  - 藍色光暈效果 `rgba(0, 102, 255, 0.15)` 營造優雅氛圍
  - 參考 [astro-chiri.netlify.app](https://astro-chiri.netlify.app/) 的設計美學

- ✍️ **字體系統優化**
  - 主字體更換為 **Inter**（與 Chiri 參考網站一致）
  - 內文字重從 300 (Light) 提升到 **400** (Regular)
  - 標題字重從 300-400 提升到 **500-600** (Medium-SemiBold)
  - 移除過淡的 300 字重，改用 400/500/600 組合
  - 文字更清晰、更易讀、更有質感

- 🔧 **文字可讀性修復**
  - 修復專案頁面標題顏色（白底白字 → 深色文字）
  - 修復專案描述透明度過低問題
  - 統一使用主題顏色變數，確保一致性

### 修改
- 📝 更新首頁標題為「2018 接觸區塊鏈，過程中點各式各樣的技能點做過 Marketing/Operations 的一個會寫點程式的資管生」
