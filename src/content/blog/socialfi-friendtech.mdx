## 什麼是 FriendTech？

FriendTech 是一個去中心化社交平台（SocialFi），透過與 Ｘ 綁定的方式，讓用戶可以使用 Base 鏈上的 ETH 購買 FT 上用戶的份額（Shares），獲取與群主直接對談的權利。

> 如果要和 Web2 的產品對標相當於 OnlyFans、知識衛星或者是任意付費加入的私有群組！

## 我該怎麼評斷 FriendTech 是否適合加入？

在回答這個問題之前，應該要確認自己究竟是為了什麼原因想加入 FT 的，簡單列了三點：

1. 空投
2. 賺錢
3. 交友

### 空投

我們可以看到 FT 的介面中有個很明顯的分類叫做 Airdrop，那麼這個產品的 Airdrop 會吸引人呢？主要源自於大家對 [Paradigm](https://www.paradigm.xyz/) 的預期，關於 Paradigm 的詳細介紹可以看幣研的介紹，簡單來說他們就是 dYdX 以及 Blur 會起來的幕後功臣！

### 賺錢

在 FT 的賺錢模式主要分成兩種：

1. 透過 Shares/Keys 賺錢，與買賣股票或加密貨幣一樣，購買潛力股並在高點賣掉。（但要記得有高額的手續費，每筆 10%）
2. 透過營銷自己的房間（Room）來吸引別人加入你的房間（Room），進而賺取交易手續費的返傭。

### 交友

交友的部分我認為就不用多說了，SocialFi 為社群/社交類型的 Finance 遊戲。

也有許多很好的案例像是 Alvin 在很多地方提到他因為 FT 認識了很多很好的朋友，像是我跟 Leo 也因為 FT 還有 TON 重新建立起一些溝通的橋樑！

## 角色定位

目前就我所認知，預期會加入 FT 遊戲的人主要分成三種：

- 沒有粉絲基礎的新手
- 有粉絲基礎的 KOL
- 區塊鏈開發者

### 一、沒有粉絲基礎的新手

給這類型的人，如果你跟我一樣，那我認為一開始可以思考一下自己是什麼類型的人。

例如說，你是一個 Researcher 但是你沒有粉絲基礎，所以你想要透過 FT 來 Build Up 一個屬於自己的私人分享群。接下來就是持續在 FT 以及 X 上經營並且多多分享自己的看法，也可以透過在 X 上去推銷自己，或者是買入更多的 Keys 去建立更多的 Connection。

**都會是建立個人品牌的好做法！**

### 二、有粉絲基礎的 KOL

若是有粉絲基礎的 KOL，會相比沒有粉絲基礎的新手更容易，一開始創建帳號時，想必會有大量機器人或是真人在刷你的 Keys，在一開始即會有一些 Fees Return，接下來你可以透過舉辦活動，或者是透過個人的影響力去吸引別人加入…等。

目前看到比較多的是 Bots Developer / Tools Developer / Researcher / Meme Catchers / 嚕毛專家 … 等

### 三、開發者

開發者就更簡單，主要也分三種：

1. MEV Bots
2. 大量購買的機器人，跟交易一樣寫策略
3. 搶活動機器人

## 如何運營 FriendTech？

### 這裡提供我所知道的運營模式，主要分幾種：

1. Alpha Group
2. Private Group
3. Campaign
4. Airdrop
5. 各式各樣職業的分享群

### FT 工具

1. **frentech.octav.fi**：可以查看地址的真實收入狀況（未實現損益）
2. **fren.trade**：可以查看線圖與積分，若購買 Founder 的 Key，可以解鎖更多功能
3. **parsec.fi**：可以透過 parsec.fi/explorer/ 填入你的地址，查看人際關係圖以及分數
4. **friendmex.com 和 openfriend.tech**：可以協助多 Key 買賣，但一樣請注意資安安全
5. **dune.com/cryptokoryo/friendtech**：FriendTech Dashboard
6. **twitter.com/x_friendtech**：可以在 X 上實時查看 FT 上 Key Info 的 Extension
